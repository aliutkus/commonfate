Common Fate Model
================

.. figure:: https://cloud.githubusercontent.com/assets/72940/13906456/402211d6-ef11-11e5-8103-12944f5404f4.png
   :alt: cfm

   cfm

An adapted factorization model similar to the PARAFAC/CANDECOMP
factorisation allows to decompose the *common fate transform* tesnor
into different time-varying harmonic sources based on their particular
common modulation profile: hence the name Common Fate Model.
